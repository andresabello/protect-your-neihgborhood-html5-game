# Protect Your Neighborhood Game
### Frameworks
Laravel, Vue, Phase 3

### Instructions
clone into local server... git clone
cp .env.example .env
composer install
php artisan key:generate
php artisan serve 
npm install 
npm run dev

go to the browser and play
