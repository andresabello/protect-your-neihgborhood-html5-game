# Protect Your Neighborhood Game
### Frameworks
Laravel, Vue, Phaser 3

### Instructions
1. clone into local server... git clone

2. cp .env.example .env

3. composer install

4. php artisan key:generate
5. php artisan serve 
6. npm install 
7. npm run dev

go to the browser and play
